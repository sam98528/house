# 001 - 프로젝트 초기 설정

**날짜**: 2026-03-30
**상태**: 진행중

## 결정 사항

### 기술 스택
- Next.js 16 (App Router) + TypeScript + Tailwind CSS v4
- shadcn/ui (컴포넌트 라이브러리)
- Supabase (DB + Auth)
- Drizzle ORM
- Auth.js v5 (카카오/네이버 소셜 로그인)
- Kakao Maps JavaScript SDK
- Vercel 배포

### 데이터 소스
- 공공데이터포털 (data.go.kr) LH 분양임대 공고 API
- 추가 조사 필요: 청약홈 API, 마이홈포털 API

### 지도
- Kakao Maps 선택 (무료 일일 30만건, 한국 주소 정확도 높음)
- Naver Maps 대비 개발자 생태계가 더 활발

## 팀 구성 (에이전트)

| 역할 | 담당 |
|------|------|
| 관리자 (PM) | 일 분배, 사용자 소통, 전체 진행 관리 |
| 부동산 전문가 | 시장 분석, 공고 데이터 구조 설계 자문 |
| 청약 제도 전문가 | 청약 자격/유형 로직, 필터 조건 설계 |
| 프론트엔드 개발자 | UI/UX 구현, 컴포넌트 개발 |
| 서버 개발자 | API 연결, DB 설계, 데이터 동기화 |
| QA | 테스트, 품질 검증, 접근성 |
